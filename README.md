# Simple Reader
Read images in a given directory

Works with .webp files

To run
``` bash
npm install
npm start
```

Use mt.exe to edit windows manifest if needed.

Sample Directory Structure
```
gallery/
├─ Album 1/
│  ├─ image.png
│  ├─ image2.png
├─ Album 2/
├─ Manga 1/
│  ├─ 1.png
│  ├─ 2.png

```
